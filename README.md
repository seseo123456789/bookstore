지금 읽자, 책(북스토어)
===
다양한 도서의 정보를 제공하는 온라인 인터넷서점을 운영한다.

🖥️프로젝트 목표
===
- 도서의 상세정보를 확인 할 수 있다.
- 선택한 도서를 장바구니에 담을 수 있다.
- 첨부파일을 업로드하며 도서리뷰를 작성 할 수 있다.

⏲️개발 기간
===
